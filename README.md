# autoVencordSourceInstaller
Automatically installs Vencord from source and copies over the plugins folder from the previous Vencord install to the new one, leaving behind a backup of the files.


# Instructions
1. Create an empty folder where you would like to install Vencord from source.
2. Install Vencord from source as normal.
3. Place the `updater.sh` file in the folder.
4. Run the file whenever you would like to download Vencord from source.

# Other information
I made this script to prevent having to manually install Vencord repeatedly after it misteriously disappears from my Discord client. It is coded by a beginner with no peer review and may have flaws. 

## Disclaimers
You must have Vencord installed from source in the folder you are running this script in **before** using it.
Use this program at your own risk. It has no form of error handling and was made specifically for my computer's setup.
Backing up your Vencord files is advised.
